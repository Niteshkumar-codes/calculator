🧮 Calculator Web App

A simple and responsive Calculator Web Application built using HTML, CSS, and JavaScript.
This calculator performs basic arithmetic operations with a clean and user-friendly interface.

🚀 Features

➕ Addition

➖ Subtraction

✖ Multiplication

➗ Division

🔢 Supports decimal numbers

🧹 AC (All Clear) button

⌫ DEL (Delete) button

⚡ Instant result using =

🎨 Modern UI design


🛠️ Technologies Used

HTML5

CSS3

JavaScript (Vanilla JS)


📂 Project Structure

calculator-project/

│
├── index.html   # Structure of calculator

├── style.css      # Styling of calculator

└── scripts.js     # Logic and functionality


💻 How to Run

Download or clone this repository

Open the project folder

Double-click on index.html

Calculator will run in your browser
